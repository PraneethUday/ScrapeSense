## ğŸ”§ Troubleshooting

**Issue: "The following errors occurred" when loading**
- Make sure you selected the `dist/` folder, not `src/`
- Refresh `chrome://extensions/` page

**Issue: Extension appears but UI doesn't show**
- Refresh the webpage after loading the extension
- Open developer console (F12) to check for errors

**Issue: Missing permissions warning**
- This is normal - the extension requests permissions to:
  - Inject UI into all pages
  - Access current page info
  - Store settings locally
